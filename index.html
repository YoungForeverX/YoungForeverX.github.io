npm全称 node package manager （node包管理器）。 npm提供一个巨大的在线仓库（registry），里面有几百万人写好并开源的“包”。 在终端（terminal）中用npm就可以下载很多有用的extension package。 但npm是国外的软件，在国内下载很慢，所以中国淘宝镜像搞了一个cnpm，用cnpm下载就很快了。
