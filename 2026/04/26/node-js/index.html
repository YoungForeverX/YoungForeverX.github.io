node.js

Created2026-04-26|Updated2026-04-28
|Post Views:

node.js 可以把javascript从浏览器中拓展到服务端。

Node.js 就是这个让 JavaScript 能够脱离浏览器、在你电脑（或服务器）上独立运行的环境。

javascript是一种语言，通常属于前端语言，和html，css共同组成一个网页。

HTML 和 CSS 基本只能待在浏览器里。但 JavaScript 借助 Node.js，已经打破了“前端”的界限。它不仅能写网页，还能写后端服务器、桌面软件甚至手机 App。

后端语言：

1 . C/C++

2 . Java

3 . Python

4 . 借助node.js的Javascript

前端语言：

  1. HTML
  2. CSS
  3. Javascript，最初叫Livescript
