私网IP

只要 IP 落在下面任意区间，就是私网 IP，其余为公网 IP：

1）A 类私网：10.0.0.0 /8

范围：10.0.0.0 ~ 10.255.255.255 用途：大型企业、工厂、机房、校园、机器人内网

2）B 类私网：172.16.0.0 /12

范围：172.16.0.0 ~ 172.31.255.255 注意：只有 172.16～172.31 才是私网，172.115、32255 都是公网

3）C 类私网：192.168.0.0 /16

范围：192.168.0.0 ~ 192.168.255.255 用途：家用路由器最常见

额外特殊保留地址（不属于公网，也不能上网）

  1. 本地回环：127.0.0.0/8（127.x.x.x，本机内部通信）
  2. 链路本地地址：169.254.0.0/16（电脑自动获取失败时分配，无法跨设备上网）
  3. 组播、保留测试段：224~255 开头，日常几乎遇不到

二、快速判断步骤（口算就能判断）

  1. 把 IP 拆成四段数字 a.b.c.d
  2. 分类判断：
  • 如果 a == 10 → 私网
  • 如果 a == 172 且 16 ≤ b ≤ 31 → 私网
  • 如果 a == 192 且 b == 168 → 私网
  • 如果 a == 127 → 本地回环
  • 如果 a == 169 且 b == 254 → 链路本地
  • 其他全部 = 公网 IP
